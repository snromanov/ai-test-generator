# Сырые требования

Эта папка предназначена для черновых требований в свободной форме.
Храните их в текстовых файлах (`.md` или `.txt`). В одном файле может быть
одно или несколько требований.

Рекомендации:
- Одно требование — один блок
- Блоки разделяйте пустой строкой или `---`
- При наличии используйте теги `[Back]`, `[Front]`, `[API]`, `[UI]`
